Lesson 2: Inheritance

Concept: OOP - Inheritance

Inheritance lets a child class reuse fields and methods from a parent class. In Java, a class inherits with the extends keyword.

What to remember

  • All classes ultimately inherit from Object.
  • Use super() to call a parent constructor.
  • Method overriding redefines parent behavior in a subclass.
  • Override methods with the same signature for polymorphism.
