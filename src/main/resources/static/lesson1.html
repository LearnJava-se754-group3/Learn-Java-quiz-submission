Lesson 1: Classes and Objects

Concept: OOP - Classes & Objects

A class is a blueprint that defines fields and behavior. An object is an instance created from that class using the new keyword.

What to remember

  • Use class to declare a class in Java.
  • Use new to create an object from a class.
  • Constructors are used to initialize object state.
  • Primitive fields like int default to 0.
